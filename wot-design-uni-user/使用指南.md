## 🚀 立即开始使用

### 步骤 1: 复制配置到项目

将 `.claude` 目录复制到你的 uni-app 项目根目录：

```bash
cp -r wot-design-uni-user/.claude /path/to/your/project/
```

### 步骤 2: 安装依赖

```bash
npm i wot-design-uni sass -D
```

**⚠️ 重要：** Sass 版本必须 <= 1.78.0

### 步骤 3: 配置 easycom

在 `pages.json` 中添加：

```json
{
  "easycom": {
    "autoscan": true,
    "custom": {
      "^wd-(.*)": "wot-design-uni/components/wd-$1/wd-$1.vue"
    }
  }
}
```

### 步骤 4: 启动开发

```bash
npm run dev:mp-weixin  # 微信小程序
# 或
npm run dev:h5         # H5
```

### 步骤 5: 开始使用

现在你可以开始使用 Wot Design Uni 组件了！

## 🔍 常见问题

### Q1: 组件无法显示？
**检查：**
1. easycom 配置是否正确
2. 组件名称是否以 `wd-` 开头
3. 修改配置后是否重新编译

### Q2: 样式如何自定义？
**方法：**
1. 使用 CSS 变量（推荐）
2. 使用 `custom-style` 属性
3. 创建自定义样式类

### Q3: 某些功能在特定平台不生效？
**原因：** 平台差异

**解决：**
1. 查看官方文档的平台差异说明
2. 使用条件编译处理
3. 测试所有目标平台

### Q4: 类型定义找不到？
**解决：**
```typescript
import type { FormInstance } from 'wot-design-uni/components/wd-form/types'
```

### Q5: 表单验证不生效？
**检查：**
1. 每个 form-item 是否有 prop 属性
2. ref 是否正确绑定
3. 是否调用了 validate() 方法

## 📊 效果对比

### 使用前
```vue
<template>
  <view class="form">
    <input v-model="username" placeholder="用户名" />
    <input v-model="password" type="password" placeholder="密码" />
    <button @click="login">登录</button>
  </view>
</template>

<script>
export default {
  data() {
    return {
      username: '',
      password: ''
    }
  },
  methods: {
    login() {
      // 手动验证
      if (!this.username) {
        alert('请输入用户名')
        return
      }
      if (!this.password) {
        alert('请输入密码')
        return
      }
      // 登录逻辑
    }
  }
}
</script>
```

**问题：**
- ❌ 没有使用组件库
- ❌ 手动实现表单验证
- ❌ 用户体验差
- ❌ 代码不规范
- ❌ 没有类型提示

### 使用后
```vue
<template>
  <wd-form ref="formRef" :model="formData" :rules="rules">
    <wd-input
      v-model="formData.username"
      label="用户名"
      prop="username"
      required
    />
    <wd-input
      v-model="formData.password"
      label="密码"
      prop="password"
      type="password"
      required
    />
    <wd-button type="primary" :loading="isLoading" @click="handleLogin">
      登录
    </wd-button>
  </wd-form>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import type { FormInstance } from 'wot-design-uni/components/wd-form/types'
import { useToast } from 'wot-design-uni'

const toast = useToast()
const formRef = ref<FormInstance>()
const isLoading = ref(false)

const formData = ref({
  username: '',
  password: ''
})

const rules = {
  username: [
    { required: true, message: '请输入用户名' }
  ],
  password: [
    { required: true, message: '请输入密码' }
  ]
}

const handleLogin = async () => {
  try {
    await formRef.value?.validate()
    isLoading.value = true
    toast.loading('登录中...')

    // 登录逻辑
    await login(formData.value)

    toast.close()
    toast.success('登录成功')
  } catch (error) {
    toast.close()
    toast.error('登录失败')
  } finally {
    isLoading.value = false
  }
}
</script>
```

**优势：**
- ✅ 使用统一的组件库
- ✅ 自动表单验证
- ✅ 优秀的用户体验
- ✅ 代码规范统一
- ✅ 完整的类型提示
