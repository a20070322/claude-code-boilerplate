# /doc - 文档维护命令

## 描述
统一的文档维护入口，支持创建、更新、检查各种类型的文档

## 使用方法
```
/doc                # 显示文档维护菜单
/doc readme         # 创建/更新 README
/doc check          # 检查文档质量
```

## 执行步骤

### 1. 显示菜单（无参数时）
列出可用的文档操作：
- 创建 README
- 更新使用指南
- 检查文档质量

### 2. README 创建/更新
**检查:**
- [ ] README.md 是否存在
- [ ] 当前是什么类型的项目

**生成内容:**
- 项目名称和简介
- 快速开始（3 步）
- 功能特性列表
- 使用示例
- 相关链接

### 3. 文档质量检查
**检查项:**
- 标题层级是否正确
- 代码块是否有语言标注
- 链接是否有效
- 是否有示例代码
- 格式是否统一

**输出:**
发现问题列表和修复建议

## 注意事项
1. 生成前先检查文件是否已存在
2. 已有内容会被保留，只更新缺失部分
3. 使用统一的格式风格
4. 确保示例代码正确
