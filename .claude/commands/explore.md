# /explore - 项目探索命令

## 描述
探索和分析项目结构，生成项目理解报告

## 使用方法
```
/explore              # 探索当前项目
/explore <路径>       # 探索指定目录
```

## 执行步骤

### 1. 扫描项目结构
- 识别主要目录
- 识别配置文件
- 识别源代码文件

### 2. 识别技术栈
- 通过配置文件识别语言
- 识别使用的框架
- 识别构建工具

### 3. 分析核心模块
- 找出入口文件
- 分析主要模块
- 识别依赖关系

### 4. 生成报告
包含:
- 项目结构
- 技术栈
- 核心模块
- 关键文件
- 下一步建议

## 输出示例

```
# 项目探索报告

## 📁 项目结构
├── src/          # 源代码
├── config/       # 配置文件
└── docs/         # 文档

## 🔧 技术栈
- 语言: JavaScript
- 框架: React 18
- 构建工具: Vite

## 🎯 核心模块
1. App.jsx - 应用入口
2. router/ - 路由配置
3. api/ - API 接口

## 💡 下一步建议
1. 查看 README 了解项目概述
2. 查看 package.json 了解依赖
3. 运行 npm start 启动项目
```
